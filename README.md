# Restaurant Landing Page

Landing page responsiva com tema de restaurante, construída com HTML5, CSS3 e JavaScript.
Apresenta design limpo, animações de scroll fluidas e interatividade moderna.

## Visão geral

- Layout totalmente responsivo, adaptado para dispositivos móveis, tablets e desktops
- Animações de scroll suaves via ScrollReveal, com carregamento progressivo dos elementos
- Interatividade com jQuery para manipulação do DOM e gerenciamento de eventos
- Tipografia e ícones otimizados com Google Fonts e Font Awesome

---

## Tecnologias

| Tecnologia | Finalidade |
|---|---|
| HTML5 | Marcação semântica e estruturada |
| CSS3 | Estilização responsiva e animações |
| JavaScript | Lógica e funcionalidades interativas |
| jQuery | Manipulação de DOM e eventos |
| ScrollReveal | Animações ativadas pelo scroll |
| Font Awesome | Biblioteca de ícones vetoriais |

## Como executar

```bash
# Clone o repositório
git clone https://github.com/seu-usuario/restaurant-landing

# Acesse a pasta do projeto
cd restaurant-landing

# Abra no navegador
open index.html
```

Nenhuma dependência ou build necessário — basta abrir o `index.html` diretamente no navegador.
